### 备注：git遇到小问题

###### git push到GitHub的时候遇到! [rejected] master -> master (non-fast-forward)的问题
```
1. git pull origin master --allow-unrelated-histories //把远程仓库和本地同步，消除差异
2. 重新add和commit相应文件
3. git push origin master
```
